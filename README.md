## 🚀 Projects

### 🍋 Limón Method V2
> *A two-phase AI-powered exam preparation pipeline for any subject, any exam board.*

Most exam prep tools make students do the heavy lifting — searching for notes, guessing what's important, hoping the practice questions are relevant. **Limón flips that.**

You drop your past papers. Claude does the rest.

**How it works:**

| Phase | What Happens |
|-------|-------------|
| **Phase 1 — Immediate Answers** | Paste or upload question papers. Claude reads every question and answers all of them. No extra prompting. |
| **Phase 2 — Say "Limón"** | Unlocks pattern analysis, structured notes built for NotebookLM, and 4 practice sets using the school repetition model. |

**Why it works:** Schools have repeated questions for decades because repetition *with variation* is how retention actually happens. Limón automates that logic — by set 4, no real exam question should catch you off guard.

- ✅ Any subject — sciences, humanities, maths, languages, social studies
- ✅ Any exam board
- ✅ Notes formatted for direct NotebookLM import
- ✅ Works as an installable Claude skill on claude.ai
